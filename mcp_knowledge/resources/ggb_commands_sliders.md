# GeoGebra 命令白名单：滑动条

同类对象只推荐下面这一种标准写法。能用白名单就不要改写成别的等价命令。

## 标准写法

- 滑动条：`t=Slider(0,1,0.01)`

## 使用说明

- 单参数路径动点优先使用 `t`
- 双参数面上动点优先使用 `u`、`v`
- 滑动条优先用于参数控制、2D 路径动点、3D 路径动点、3D 面上参数动点
- 2D 路径动点应先创建滑动条，再写 `P=Point(path,t)`
- 3D 路径动点应先创建滑动条 `t`，再写 `P=Point(path,t)`
- 3D 面上参数动点应先创建滑动条 `u`、`v`，再写 `P=(x(u,v),y(u,v),z(u,v))`
