# GeoGebra 命令白名单：2D 圆锥曲线

同类对象只推荐下面这一种标准写法。能用白名单就不要改写成别的等价命令。

## 标准写法

- 椭圆：`ellipse1=Ellipse(F1,F2,P)`
- 双曲线：`hyper1=Hyperbola(F1,F2,P)`
- 抛物线：`para1=Parabola(F,d)`

## 使用说明

- 椭圆、双曲线必须先创建两个焦点，再创建一个已知曲线上点，最后创建图像
- 抛物线必须先创建焦点与准线，最后创建图像
- 圆锥曲线对象也必须显式命名；题目未给名时，优先使用 `ellipse1`、`hyper1`、`para1`
- 依赖对象必须先出现，再创建圆锥曲线
