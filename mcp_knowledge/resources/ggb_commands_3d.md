# GeoGebra 命令白名单：3D 基础对象与动点

同类对象只推荐下面这一种标准写法。能用白名单就不要改写成别的等价命令。

## 标准写法

- 点：`A=(0,0,0)`
- 直线：`lineAB=Line(A,B)`
- 圆柱：`cyl1=Cylinder(lineAB,1)`
- 圆锥：`cone1=Cone(lineAB,1)`
- 3D 路径动点：`P=Point(path,t)`
- 3D 面上参数动点：`P=(x(u,v),y(u,v),z(u,v))`

## 使用说明

- 先创建三维点，再创建三维直线或实体
- 圆柱、圆锥依赖的轴对象必须先创建
- 3D 路径动点先创建路径对象，再创建滑动条 `t`，最后写 `P=Point(path,t)`
- 3D 面上参数动点先创建两个滑动条 `u`、`v`，再写 `P=(x(u,v),y(u,v),z(u,v))`
- 如果题目还要求显示对应曲面或实体，应额外显式创建该曲面或实体对象
- 不要写成 `Point(surface,u,v)` 这种未纳入白名单的形式
- 如果题目本质上是线框关系图，优先保留基础点和线
